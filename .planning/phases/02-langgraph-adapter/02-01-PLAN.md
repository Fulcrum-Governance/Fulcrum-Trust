<objective>
Implement TrustAwareGraph — the LangGraph adapter that wraps any uncompiled StateGraph with automatic trust evaluation at node boundaries, including hard graph termination via conditional edges when trust falls below threshold.

Purpose: Fulfils the core value proposition of Phase 2: developers hand their existing StateGraph to TrustAwareGraph and get circuit-breaking trust evaluation for free, with zero changes to their graph code. The wrapped graph terminates automatically (routes to END) when trust is below threshold — not just fires a callback.

**Add TrustAwareGraph to fulcrum_trust/adapters/langgraph.py** (append after CallbackRegistry and OutcomeClassifier):

Class `TrustAwareGraph`:

```
__init__(self, graph: "StateGraph", trust_manager: TrustManager,
         agent_a: str = "orchestrator", agent_b: str = "worker") -> None
```
- If `not _LANGGRAPH_AVAILABLE:` raise `ImportError("langgraph is required for TrustAwareGraph. Install with: pip install 'fulcrum-trust[langgraph]'")`
- Store: `self._graph`, `self._trust_manager`, `self._agent_a`, `self._agent_b`
- Create: `self._classifier = OutcomeClassifier()`, `self._callbacks = CallbackRegistry()`
- Store previous termination state: `self._was_terminated = False`

Callback registration methods (LANG-04):
```python
def on_trust_change(self, callback: Callable[[TrustState], None]) -> None: ...
def on_circuit_break(self, callback: Callable[[TrustState], None]) -> None: ...
def on_recovery(self, callback: Callable[[TrustState], None]) -> None: ...
```
Each appends to the corresponding `self._callbacks` list.

Internal `_make_node_wrapper(self, node_fn: Callable[[Any], Any]) -> Callable[[Any], Any]`:
- Detect async: `if asyncio.iscoroutinefunction(node_fn):`
  - Produce `async def wrapped(state: Any) -> Any:` using `@functools.wraps(node_fn)`
  - Body: `result = await node_fn(state)` → call `self._evaluate_and_route(state, result)` → `return result`
- Else: produce sync `def wrapped(state: Any) -> Any:` using `@functools.wraps(node_fn)`
  - Body: `result = node_fn(state)` → call `self._evaluate_and_route(state, result)` → `return result`
- Return the appropriate wrapper.

Internal `_evaluate_and_route(self, prior_state: Any, result: Any) -> None`:
- `outcome = self._classifier.classify(prior_state, result)`
- `trust_state = self._trust_manager.evaluate(self._agent_a, self._agent_b, outcome)`
- Fire `self._callbacks.fire_trust_change(trust_state)` always.
- `is_terminated = self._trust_manager.should_terminate(self._agent_a, self._agent_b)`
- If `is_terminated`: fire `self._callbacks.fire_circuit_break(trust_state)`
- If `not is_terminated and self._was_terminated`: fire `self._callbacks.fire_recovery(trust_state)`
- `self._was_terminated = is_terminated`

`compile(self, **kwargs: Any) -> Any` (LANG-01 + LANG-03):

**Node wrapping strategy** — mutate node callables in the builder before compilation. In LangGraph 0.2.x each entry in `builder.nodes` is a `StateNodeSpec` with a `.runnable` field (a `RunnableLambda`). Access the underlying callable via `.runnable.func`. Replace by calling `spec.runnable = RunnableLambda(wrapped_fn)`. Use a try/except for version-drift resilience:

```python
for name, spec in list(self._graph.nodes.items()):
    if name == "__start__":
        continue
    try:
        original_fn = spec.runnable.func  # type: ignore[union-attr]
        wrapped = self._make_node_wrapper(original_fn)
        from langchain_core.runnables import RunnableLambda  # type: ignore[import]
        spec.runnable = RunnableLambda(wrapped)  # type: ignore[union-attr]
    except AttributeError:
        if callable(spec):
            wrapped = self._make_node_wrapper(spec)  # type: ignore[arg-type]
            self._graph.nodes[name] = wrapped  # type: ignore[index]
```

**Automatic termination via conditional edges (LANG-03):**

After wrapping all nodes, inject a conditional edge for each non-`__start__` node so that the graph routes to `END` when `trust_manager.should_terminate()` returns True. This is the hard-termination guarantee: the graph itself stops executing, not just a callback firing.

For each wrapped node name, determine the normal next node by inspecting `self._graph.edges` (a dict mapping source → destination in 0.2.x). If the source has a normal edge to another node (not END), replace that edge routing with a conditional edge:

```python
from langgraph.graph import END as _END

# Build a map of existing simple edges: source_node -> target_node
# In LangGraph 0.2.x, self._graph.edges is a set of (src, dst) tuples.
# Convert to dict for lookup (last edge wins if multiple, which is unusual).
edge_map: dict[str, str] = {}
for edge in self._graph.edges:
    src, dst = edge  # each edge is a (str, str) tuple
    edge_map[src] = dst

node_names = [n for n in self._graph.nodes if n != "__start__"]

for node_name in node_names:
    normal_next = edge_map.get(node_name)
    if normal_next is None:
        # No simple outgoing edge (may already be conditional). Skip.
        continue

    # Capture variables for the closure.
    _normal_next = normal_next
    _agent_a = self._agent_a
    _agent_b = self._agent_b
    _tm = self._trust_manager

    def _make_routing_fn(agent_a: str, agent_b: str, normal_next: str) -> Callable[[Any], str]:
        def routing_fn(state: Any) -> str:
            """Route to END if trust is below threshold, else continue normally."""
            if _tm.should_terminate(agent_a, agent_b):
                return "terminate"
            return "continue"
        return routing_fn

    routing_fn = _make_routing_fn(_agent_a, _agent_b, _normal_next)

    # Remove the existing simple edge before adding the conditional one.
    # In LangGraph 0.2.x, edges is a set; discard the tuple.
    self._graph.edges.discard((node_name, _normal_next))

    # path_map MUST include END (as the string key "terminate") or LangGraph raises ValueError.
    self._graph.add_conditional_edges(
        node_name,
        routing_fn,
        {"continue": _normal_next, "terminate": _END},
    )
```

Note: `self._graph.edges` is a `set` in LangGraph 0.2.x. If the version uses a different internal structure, the `discard` call will be a no-op and the conditional edge will be added alongside (LangGraph will use the conditional edge and ignore the simple edge at routing time — acceptable fallback). If `edges` is not a set (AttributeError), wrap in try/except and skip the discard step.

After injecting conditional edges, compile:
```python
return self._graph.compile(**kwargs)
```

The complete `compile()` method should:
1. Wrap all node callables (trust evaluation fires inside each node).
2. Inject conditional edges for END routing (LANG-03 hard termination).
3. Call and return `self._graph.compile(**kwargs)`.
