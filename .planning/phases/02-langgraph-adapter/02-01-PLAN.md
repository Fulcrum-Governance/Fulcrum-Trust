**Add TrustAwareGraph to fulcrum_trust/adapters/langgraph.py** (append after CallbackRegistry and OutcomeClassifier):

Class `TrustAwareGraph`:

```
__init__(self, graph: "StateGraph", trust_manager: TrustManager,
         agent_a: str = "orchestrator", agent_b: str = "worker") -> None
```
- If `not _LANGGRAPH_AVAILABLE:` raise `ImportError("langgraph is required for TrustAwareGraph. Install with: pip install 'fulcrum-trust[langgraph]'")`
- Store: `self._graph`, `self._trust_manager`, `self._agent_a`, `self._agent_b`
- Create: `self._classifier = OutcomeClassifier()`, `self._callbacks = CallbackRegistry()`
- Store previous termination state: `self._was_terminated = False`

Callback registration methods (LANG-04):
```python
def on_trust_change(self, callback: Callable[[TrustState], None]) -> None: ...
def on_circuit_break(self, callback: Callable[[TrustState], None]) -> None: ...
def on_recovery(self, callback: Callable[[TrustState], None]) -> None: ...
```
Each appends to the corresponding `self._callbacks` list.

Internal `_make_node_wrapper(self, node_fn: Callable[[Any], Any]) -> Callable[[Any], Any]`:
- Detect async: `if asyncio.iscoroutinefunction(node_fn):`
  - Produce `async def wrapped(state: Any) -> Any:` using `@functools.wraps(node_fn)`
  - Body: `result = await node_fn(state)` → call `self._evaluate_and_route(state, result)` → `return result`
- Else: produce sync `def wrapped(state: Any) -> Any:` using `@functools.wraps(node_fn)`
  - Body: `result = node_fn(state)` → call `self._evaluate_and_route(state, result)` → `return result`
- Return the appropriate wrapper.

Internal `_evaluate_and_route(self, prior_state: Any, result: Any) -> None`:
- `outcome = self._classifier.classify(prior_state, result)`
- `trust_state = self._trust_manager.evaluate(self._agent_a, self._agent_b, outcome)`
- Fire `self._callbacks.fire_trust_change(trust_state)` always.
- `is_terminated = self._trust_manager.should_terminate(self._agent_a, self._agent_b)`
- If `is_terminated`: fire `self._callbacks.fire_circuit_break(trust_state)`
- If `not is_terminated and self._was_terminated`: fire `self._callbacks.fire_recovery(trust_state)`
- `self._was_terminated = is_terminated`

`compile(self, **kwargs: Any) -> Any` (LANG-01 + LANG-03):

Implementation strategy — reconstruct a new StateGraph to avoid StateNodeSpec internals (RESEARCH.md Pitfall 2):

1. Get the state schema from `self._graph` — access `self._graph.schema` (the TypedDict class passed to `StateGraph(schema)`). Store it.
2. Enumerate node names: `node_names = list(self._graph.nodes.keys())`. Filter out `"__start__"` if present.
3. Require the user to have passed node callables via the `StateGraph.add_node()` calls — access them from `self._graph.nodes`. In LangGraph 0.2.x, `StateGraph.nodes` is a `dict[str, Any]` where each value is a `RunnableLambda`. Extract the original callable via `.func` attribute if present, else use the value directly. Store wrapped callables: `wrapped = {name: self._make_node_wrapper(fn) for name, fn in ...}`

IMPORTANT: Do NOT attempt to reconstruct edges — this requires introspecting internal `self._graph.edges` and `self._graph.branches` structures that are unreliable (RESEARCH.md Open Question 2). Instead use the simpler trust-injection strategy:

Inject trust routing using `interrupt_after` on every node at compile time. After each node runs, LangGraph will surface control flow back to the wrapper. BUT — this is too complex for Phase 2.

**Simpler compile() strategy (approved):** Wrap node functions in-place on the original builder by using `builder.nodes` direct mutation. In LangGraph 0.2.x each entry in `builder.nodes` is a `StateNodeSpec` with a `.runnable` field (a `RunnableLambda`). Access the underlying callable via `.runnable.func`. Replace by calling `builder.nodes[name] = dataclasses.replace(spec, runnable=RunnableLambda(wrapped_fn))`.

If `.runnable.func` or `RunnableLambda` access fails at runtime (version drift), fall back to: create a new `StateGraph(self._graph.schema)`, re-add nodes with wrapped callables (requires user to have passed callables separately via `node_callables` dict). Since the simpler approach may be brittle, implement it defensively:

```python
def compile(self, **kwargs: Any) -> Any:
    """Wrap all nodes with trust evaluation and compile.

    Mutates node callables in the builder before compilation.
    Uses try/except to handle StateNodeSpec internal API differences
    across LangGraph 0.2.x patch versions.
    """
    for name, spec in list(self._graph.nodes.items()):
        if name == "__start__":
            continue
        try:
            # LangGraph 0.2.x: StateNodeSpec.runnable is a RunnableLambda
            original_fn = spec.runnable.func  # type: ignore[union-attr]
            wrapped = self._make_node_wrapper(original_fn)
            from langchain_core.runnables import RunnableLambda  # type: ignore[import]
            spec.runnable = RunnableLambda(wrapped)  # type: ignore[union-attr]
        except AttributeError:
            # Fallback: spec itself may be callable
            if callable(spec):
                wrapped = self._make_node_wrapper(spec)  # type: ignore[arg-type]
                self._graph.nodes[name] = wrapped  # type: ignore[index]
    return self._graph.compile(**kwargs)
```

Note: trust-based routing (LANG-03 circuit breaking) works because `_evaluate_and_route` is called inside every wrapped node. When trust falls below threshold, `should_terminate()` returns True and the circuit-break callback fires. The graph will continue to compile and run — actual early termination in LangGraph requires conditional edges. Add a single post-compile note in the docstring: for hard termination (stopping the graph mid-run), the user should check `trust_manager.should_terminate()` in their own conditional edge or use `TrustAwareGraph` with graphs that already have termination conditions. Phase 2 guarantees: (1) trust evaluation fires at every node, (2) circuit-break callback fires, (3) the graph CAN be structured to terminate via conditional edges the user adds. This is the correct scope.
